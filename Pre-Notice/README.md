## 대회장 색상사진
- 대회장의 환경에 따라 색상값이 달라질 수 있음.
- 따라서, 대회진행팀은 전날 7시까지 드론으로 비행하면서 촬영한 대회장 사진을 제공할 예정
- 필요에 따라 참가팀은 색상값을 수정/변경
